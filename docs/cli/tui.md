# TUI Dashboard

The `tui` command launches an interactive terminal user interface for managing GroundControl.

> **Preview:** The TUI is currently in preview and under active development. It does not yet support all features available via the CLI commands. For full functionality, use the individual CLI commands documented in the other reference pages.

## Usage

```bash
groundcontrol tui
```

This is also the default command -- running `groundcontrol` with no arguments launches the TUI.

## Description

The TUI provides a visual, keyboard-driven interface for browsing and managing GroundControl resources. It is intended as a convenient alternative to the individual CLI commands for interactive use.
